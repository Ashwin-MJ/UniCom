Help - Add Individual Page

Here you can add a single feedback for the selected student


  • First, select a category from the list available. If you want to add a new category then click the plus icon.
  • After selecting a category, a list of messages relevant to that category will be displayed. Select one of these or create a new message by clicking the relevant button.
  • You have the option of entering an optional message, however this can be kept blank.
  • Finally, select a score from 1-5 for this feedback and click "Submit & Finish"
  • If you want to immediately give this student another feedback then click "Submit & Add Another Feedback".
  • On the right you will see that students total score (across all courses) as well as recent feedback they have received. This allows you to view what other lecturers/students are giving to that student and determine what you wish to tell them accordingly.
  • Note that on "Courses" page, if you selected more than one student and chose to give Individual Feedback, then upon clicking "Submit & Finish" the page will be reloaded for the next student in the list. When all students have been given feedback you are redirected to a page which shows all feedback you have given.
